# Security Policy

## Supported Version

The latest published version of ClearCopy receives security fixes.

## Reporting a Vulnerability

Do not include sensitive URLs, authentication tokens, or personal information in
a public issue.

For non-sensitive security concerns, open a
[GitHub issue](https://github.com/sonu07-star/clearcopy-firefox/issues).

For sensitive reports, contact the maintainer privately through the support email
listed on the Mozilla Add-ons page.
