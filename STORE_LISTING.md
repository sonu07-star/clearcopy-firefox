## Summary

Remove tracking parameters and copy clean, shareable links in one click.

## Description

Share the page, not the tracking clutter.

ClearCopy removes common marketing and social tracking parameters from URLs
before you copy them. It preserves useful URL parameters, runs entirely inside
Firefox, and never sends your links anywhere.

Features:

- Clean and copy the current page from the toolbar
- Right-click any link and choose "Copy clean link"
- Use Alt+Shift+C to clean and copy the current page instantly
- See exactly which tracking parameters were removed
- Automatically recognize common new click and affiliate tracking identifiers
- Preserve useful search, product, filter, pagination, and timestamp parameters
- Replace a messy address with its clean version
- Private local-only usage totals
- No accounts, analytics, ads, or data collection

Permissions are intentionally minimal. ClearCopy does not request access to all
websites or browsing history.
